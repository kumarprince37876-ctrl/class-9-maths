1. Introduction

An Arithmetic Progression (AP) is a sequence of numbers in which the difference between consecutive terms is constant.

Example: 2, 5, 8, 11, ... (Common difference d = 3)

2. General Form of AP

If the first term is a and the common difference is d, then the nth term is:

Tn = a + (n-1)d

3. Sum of n Terms

Formula for sum of first n terms:

Sn = n/2 [2a + (n-1)d] or Sn = n/2 [a + Tn]

Where, Tn = nth term of the AP

4. Example 1

Find the 10th term of AP: 3, 7, 11, ...

Solution: a=3, d=4 â†’ T10 = 3 + (10-1)4 = 3 + 36 = 39

5. Example 2 (Sum)

Find sum of first 15 terms of AP: 2, 5, 8, ...

Solution: a=2, d=3, n=15

Sn = 15/2 [2*2 + (15-1)*3] = 7.5 [4 + 42] = 7.5*46 = 345

6. Properties

  • The common difference can be positive or negative.
  • AP can be finite or infinite.
  • Sum formula works for consecutive terms starting from first term.

7. Important Formulas

  • nth term: Tn = a + (n-1)d
  • Sum of n terms: Sn = n/2 [2a + (n-1)d]
  • Alternate sum: Sn = n/2 [a + Tn]
